# BB.Servo.Robotis

BB integration for driving RC servos via Robotis/Dynamixel over serial.

## Installation

Add `bb_servo_robotis` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:bb_servo_robotis, "~> 0.3.0"}
  ]
end
```
